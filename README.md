#MFO Admin

Admin UI and API for MakerFaire Orlando

##Dependencies

Node >= v0.12.0
Nodemon (optional)
Gulp
Bower
NPM


##Getting Started
UI
```bash
$ cd public
$ npm install && bower install
$ gulp serve
```
API
```bash
$ npm install
$ nodemon --harmony index.js
```
Note: API runs build version of app `$ cd public && gulp build`
